# Copernicus_Project
State of the art & gap analysis of the training offer in Earth Observation data using AI, webscrapping and benchmarking

# Python installation and running the script

* Download the Anaconda installer at Anaconda website, and follow the steps according your
operating system(e.g. Windows, Linux or macOS) to install it in your pc.

* Once you have installed Anaconda, open an Anaconda Prompt (e.g. Terminal/CMD), and
go the workfile path folder where you downloaded the code (e.g. cd C:\Franz\UCA M2\ Thales\code\Copernicus Project)

* Make sure you have the file copernicus env.yml in your workfile and then enter the following
command in the prompt window to create the environment for our project with all the necessary libraries:
```
conda env create âˆ’f copernicus env.yml
```
This step might takes some minutes according your internet conection.

* Now, you have to activate the new environment with the following code in the prompt:
```
conda activate web scr
```

* Running the python script to obtain the indicators:
```
python indicators.py
```
